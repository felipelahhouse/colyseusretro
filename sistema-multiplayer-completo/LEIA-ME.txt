## 🚀 Descompactar

### Windows
1. Clique com botão direito no arquivo `.tar.gz`
2. Escolha "Extrair aqui" (use 7-Zip ou WinRAR)

### Mac/Linux
```bash
tar -xzf sistema-multiplayer-completo.tar.gz
```

## 📖 Depois de Extrair

1. **Abra**: `INDEX.md` (tem tudo!)
2. **Leia**: `COMECE_AQUI.md` (instalação rápida)
3. **Instale**: `npm install peerjs firebase lucide-react`
