🪟 PASSO A PASSO NO WINDOWS:

1️⃣ BAIXAR O ARQUIVO
   • Clique no link para baixar o .zip
   • Salve em uma pasta (ex: Downloads)

2️⃣ EXTRAIR COM WINRAR
   
   OPÇÃO A - Botão Direito:
   • Clique com BOTÃO DIREITO no arquivo .zip
   • Escolha "Extrair aqui" ou "Extract here"
   • Pronto! ✅

   OPÇÃO B - Abrir com WinRAR:
   • Dê duplo-clique no arquivo .zip
   • Clique em "Extract To" (Extrair para)
   • Escolha a pasta destino
   • Clique "OK"
   • Pronto! ✅

3️⃣ ABRIR A PASTA EXTRAÍDA
   • Procure a pasta: "sistema-multiplayer-completo"
   • Ou os arquivos soltos (se escolheu "Extrair aqui")

4️⃣ ABRIR A DOCUMENTAÇÃO
   • Procure o arquivo: INDEX.md
   • Abra com Notepad, VS Code, ou qualquer editor
   • Leia as instruções! 📖
